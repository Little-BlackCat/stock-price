# Stock Price Taker

## Project Description

Stock Price Taker is a real-time stock price tracking web application built using Python and Flask for the backend, and JavaScript for the frontend.

## Technologies Used

- **Backend:**
  - Python
  - Flask

- **Frontend:**
  - HTML
  - JavaScript
  - CSS
